- Splitter
- Switcher
- TabPanel
- Text
- Textarea

## Development

- Clone the repo with `--recursive` option for cloning submodules
- Execute `/build/index.php` to make the build. Click `Build` at the page top right corner
- Find the built files in the `/lib` folder
